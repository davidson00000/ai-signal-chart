## 🗒 notes フォルダの役割

このフォルダは **構造化される前の一次メモ置き場** とする。

- 実験ログ
- 思いつきメモ
- まだ specs / strategies / reports に昇格していないアイデア

などを自由に置いてよい。

後で内容が固まったものは、必要に応じて：

- `docs/specs/`（正式なルール・仕様）
- `docs/strategies/`（再利用可能な戦略仕様）
- `docs/reports/`（検証レポート）

のいずれかに「昇格」させる。  
それまでは、ここに雑多に置いてOKという前提で運用する。
